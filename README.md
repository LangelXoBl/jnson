# Jnson

**Jnson** es una aplicación web diseñada para facilitar el trabajo con texto en formato JSON. Ofrece una serie de herramientas prácticas para formatear, editar y explorar datos JSON de manera interactiva y eficiente.

## Características principales

- **Formateo**: Mejora la legibilidad del JSON con un solo clic.
- **Edición de keys**:
  - Añade nuevas claves fácilmente.
  - Elimina claves innecesarias de forma intuitiva.
- **Visualización interactiva**: Navega y comprende estructuras JSON complejas con una interfaz amigable y dinámica.

## Tecnologías utilizadas

- **Frontend**:
  - [Vue 3](https://vuejs.org/)
  - [shadcn-vue](https://github.com/shadcn/shadcn-vue)
  - [Tailwind CSS](https://tailwindcss.com/)

## ¿Cómo usar la aplicación?

1. Sube tu archivo JSON o pega el contenido directamente en el editor.
2. Utiliza las herramientas disponibles para formatear, editar o explorar el JSON.
3. Descarga el JSON modificado o copia el contenido directamente.

## Instalación local

Si deseas probar el proyecto en tu máquina:

1. Clona este repositorio:

   ```bash
   git clone https://github.com/tu-usuario/jnson.git
   ```

2. Instala las dependencias:

   ```bash
   npm install
   ```

3. Clona este repositorio:
   ```bash
   npm run dev
   ```
